# Module: embeds

**Status:** planned — future phase

## Overview

Generates embeddable `<iframe>` and `<script>` snippets for integrating finished pieces into external websites.

## Planned scope

- Generate iframe embed code
- Generate JS embed snippet
- Embed-level analytics
- Allowed-domain allowlist per embed
- Embed versioning (pin to snapshot vs. always-latest)

## Dependencies

- PostgreSQL (Fase 2)
- `publishing` module

## Notes

This module is intentionally empty in Phase 1.
