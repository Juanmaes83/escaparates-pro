# Module: publishing

**Status:** planned — future phase

## Overview

Generates and manages public viewer URLs for finished pieces. The viewer is a read-only, client-facing render — no editor panels.

## Planned scope

- Publish a project snapshot to a stable URL
- Unpublish / revoke URL
- Password-protected publish
- Custom slug support
- View analytics (impressions, unique visitors)

## Dependencies

- PostgreSQL (Fase 2)
- `projects` module
- CDN / edge delivery

## Notes

This module is intentionally empty in Phase 1.
