# Module: projects

**Status:** planned — future phase

## Overview

Projects are the top-level creative containers inside a workspace. Each project holds one or more scenes with effects, media, and export settings.

## Planned scope

- CRUD for projects
- Duplicate / archive project
- Project-level metadata (name, thumbnail, tags)
- Version history (snapshots)

## Dependencies

- PostgreSQL (Fase 2)
- `workspaces` module

## Notes

This module is intentionally empty in Phase 1.
