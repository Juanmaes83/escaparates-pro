# Module: presets

**Status:** planned — future phase

## Overview

Reusable effect configurations that users can save, share, and apply across projects.

## Planned scope

- Save current effect config as a named preset
- List / search presets (personal + public gallery)
- Apply preset to a scene
- Fork / remix a public preset

## Dependencies

- PostgreSQL (Fase 2)
- `projects` module

## Notes

This module is intentionally empty in Phase 1.
