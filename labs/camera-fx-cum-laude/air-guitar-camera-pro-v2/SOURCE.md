# Air Guitar Camera PRO V2

Objetivo de esta version:

- Fase 1: juego claro con mision, semaforo, coach, score, combo y ranking.
- Fase 2: selector de escenarios 360 generados para punto de vista de cantante.
- Fase 3: Studio Mode. Graba eventos musicales, permite escuchar el riff y descargar WAV/eventos.

Fuentes y patrones usados:

- `Juanmaes83/7eb1cd2d4179a81e832da1236f5734dc` para el concepto Air Guitar por webcam.
- `Jeffrey-Luszcz/videotheremin` para el patron estable MediaPipe Hands + Web Audio API.
- Escenarios 360 generados para Escaparates Pro, sin logos ni texto.

Nota:

Esta carpeta es nueva e independiente. No sustituye `air-guitar-camera-pro` ni `air-guitar-camera-pro-festival`.
